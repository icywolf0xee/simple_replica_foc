原使用ch32作为mcu驱动BLDC+FOC，因文件丢失，现为复刻，本是双路电机简化成单路，因为ch32使用库文件与stm32标准库相同，所以软件使用stm32复制。

补充：ch32f307换stm32f103进行项目全体回忆重构,因为回忆项目，故错误难免，使用时需要注意依实修改

硬件：
1.电源：
    mos控制MCU开关，第一个mos为保护电路，电阻熔断断路，mos截止，第二个减少打火损耗

2.FOC驱动部分：
    1mr电阻两端SNA，SPA，采样foc桥臂导通时的电流获得相电流信息
    电源电压采样，确定电源输入规格（计划使用S3或S2），结合50mr电阻采样电流确定电机是否有出现卡死等。

3.驱动芯片选择：
    当时设计两套方案，①fd6288+3半桥，②drv8313；考虑到不量产且当时设计为电机专用所以接口富裕，故最后选择fd6288，修改mos参数以便多种适配。
